Quiz Solution

1) What does CSS stand for?
Ans:- Cascading Style Sheets

2) How do you insert a comment in a CSS file?
Ans:- /* this is a css comment */

3) Which HTML tag is used to define an internal style sheet?
Ans:- <style>

4) Which is the correct CSS syntax?
Ans:- body {color: blue;}

5) Which property is used to change the text color of an element?
Ans:- color:

6) How do you display hyperlinks without an underline?
Ans:- a {text-decoration:none;}

7) Some HTML elements have multiple states, like the link, how do you give it a particular style when the user hovers over the element?
Ans:- :hover

8) The id selector is used to specify a style for
Ans:- single, unique element

9) Which css property you will use if you want to add some margin between a DIV's border and its inner text ?
Ans:- padding

10) What property would you use to set an image instead of a standard bullet in a list?
Ans:- list-style-image:
