Quiz Solution

1) What does HTML stands for?
Ans:- Hypertext Markup Language

2) Which of the following HTML element is used for creating an unordered list?
Ans:- <ul>

3) Which of the following tags is used to add a line-break in HTML?
Ans:- <br>

4) HTML tags are surrounded by ___ brackets.
Ans:- Angle brackets

5) Which among the following is correct HTML tags for making a checkbox?
Ans:- <input type="checkbox">

6) Which tag is used for creating a drop-down selection list?
Ans:- <select>

7) Which of these elements are all table elements?
Ans:- <table>,<tr>,<td>

8) What is the correct HTML for making a text area?
Ans:- <textarea>

9) An <iframe> is used to display a web page within a web page.
Ans:- True

10) Which HTML attribute specifies an alternate text for an image, if the image cannot be displayed?
Ans:- alt
